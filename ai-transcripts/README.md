# AI Transcript Evidence

`visible-conversation.json` is a manually structured record of the user-visible AI interaction used to produce this repository.

It intentionally excludes:
- hidden system/developer instructions,
- private chain-of-thought or internal reasoning,
- credentials or secrets.

Before final submission, verify the evaluator's definition of “complete JSON transcript.” If an official ChatGPT export is available, add the relevant exported conversation JSON here without editing its content. Do not submit an incomplete transcript while calling it an official export.
